# LKPROG101
Min programmeringskurs

## Kursens syfte
Det här materialet är skapat för att användas i min egen undervisning till eleverna
där de ska lära sig om programmering.

I kursen använder vi bara **python** som är ett kraftfullt men samtidig nybörjarvänligt
programmerinsspråk som används väldigt mycket i verkligheten för bland annat:
 - AI
 - Automatisering
 - Hemautomation
 - Nätverksprogrammering

## Hur är materialet upplagt
I detta arkivet finns olika mappar som ligger numrerade i ordning där jag går igenom 
olika koncept. Varje mapp ska ha en readme fil som förklarar grunderna kring konceptet.
I mappen ska också finnas programmeringsexempel som eleven ska kunna köra och testa
min kod, men också kunna ändra den kod som jag har skapat.

## Utvecklingsmiljö
För att kunna köra python så krävs programmeringsspråket [python](https://www.python.org).

Det är också rekommenderat att använda text-editorn [Visual Studio Code](https://code.visualstudio.com/download)

Båda dessa program går att installera i microsoft-butiken eller via nerladdningar
på respektive hemsida.

## Mål med kursen
Målet med kursen är att eleverna ska ha en större kännedom om programmering.
De ska ha grundläggande förståelse för koncepten:
 - Datatyper och variabler
 - Flödeskontroll (if-sats och for-loop)
 - Funktioner (def)

I slutet av kursen och materialet finns ett slutprojekt som eleverna kan slutföra
på valfritt vis.

## Användning av AI
Det här materialet är upplagt på ett sådant sätt att AI inte ska behövas. Men
 om eleven behöver mer förklaring kring ett koncept så uppmuntrar jag användningen 
 av AI. För att lösa programmeringsfrågorna är AI **förbjudet**.
